# TOOLS.md - 本地工具笔记

技能定义“工具能做什么”。这里记录**你本地环境的具体细节**。

## 适合放这里的内容

例如：

- 摄像头名称与位置
- SSH 主机与别名
- TTS 的偏好声音
- 扬声器/房间名称
- 设备昵称
- 任何环境相关的细节

## 示例

```markdown
### 摄像头

- living-room → 客厅主视角，180° 广角
- front-door → 门口，触发报警

### SSH

- home-server → 192.168.1.100, user: admin

### TTS

- Preferred voice: "Nova" (warm, slightly British)
- Default speaker: Kitchen HomePod
```

## 为什么单独存放

技能是通用的；你的环境是独特的。分离能避免泄露基础设施，也便于更新技能。
